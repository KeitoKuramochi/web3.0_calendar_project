# Issue #3: 個人プロフィール登録画面の実装 (feature/profile-page)

## 📌 概要
学生や教員、職員が自分のプロフィールを管理・登録する画面を実装します。入力した情報はブラウザの `localStorage` に保存し、他の日程調整・リクエスト画面と連動させます。

## 🎯 ゴール
- `src/app/profile/page.tsx` および関連する CSS Module を作成し、美麗なプロフィール画面を構築すること
- 以下の入力が行えること：
  - 名前、所属、メールアドレス
  - 役割（学生、教員、職員、TA）の選択
  - 相談可能テーマ（ポチポチチェックボックス）
  - 会いやすい時間・避けたい時間の自由文入力
  - メールで入れてほしい情報
- 保存ボタンを押したときにローカルストレージへ保存され、トースト等のマイクロインタラクションで完了が通知されること
- 保存したデータが次回アクセス時に自動で入力欄に復元されること

## 🛠️ 作業手順
1. `src/app/profile/page.tsx` および `src/app/profile/profile.module.css` を作成する
2. 画面レイアウトをグラスモーフィズム調のデザインでコーディング
3. 状態管理（`useState`）と `localStorage` への永続化ロジックの実装
4. 保存時のトーストアニメーション等の実装
5. 動作確認を行い、コミット・マージする
