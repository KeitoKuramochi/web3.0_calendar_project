# Issue #6: 日程スコアリング・プライバシー表示の実装 (feature/schedule-scoring)

## 📌 概要
選択された相談相手の「曖昧な予定感」と、自分の「空き時間枠」を突合し、各時間枠の調整可能性をスコアリング表示します。また、具体的な予定理由は隠し、ぼかした表現に変換します。

## 🎯 ゴール
- 選択された相手のプロフィールを `analyzeProfile` AIに通して、好ましい時間・避けるべき時間のタグを抽出・整理すること
- 自分の空き時間リストそれぞれに対して `scoreTimeSlots` AIを実行し、◎（高）, ○（中）, △（低）, ×（不可）の判定を視覚的にわかりやすく（色分けやアイコンで）表示すること
- 調整が難しい・不可の理由について、`sanitizeReason` AIを通してプライバシーを保護し、「相手の定例ミーティング時間帯と重なる可能性があります」等のぼかした表現で表示すること
- 日程を選択し、「メール生成に進む」ボタンを押した際、選択した日時と相談先情報をローカルストレージ（またはContext）に保存して `/mail` 画面に遷移すること

## 🛠️ 作業手順
1. `src/app/match/page.tsx` に日程調整スコアリングのUIとロジックを追加
2. 日程スコアリングAI (`scoreTimeSlots`)、プロフィール解析AI (`analyzeProfile`)、プライバシー調整AI (`sanitizeReason`) の統合
3. スコアに応じたグラフィカルなバッジ表示（◎＝Emerald、○＝Blue、△＝Amber、×＝Red）の実装
4. 遷移用ボタンの実装
