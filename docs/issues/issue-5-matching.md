# Issue #5: 相談先候補マッチング機能の実装 (feature/matching)

## 📌 概要
リクエスト内容（キーワードや自由文）から、最適な相談相手（教員、キャリアセンター、TAなど）を適合度（％）付きでソート表示する画面を実装します。

## 🎯 ゴール
- `src/app/match/page.tsx` および `src/app/match/match.module.css` の作成
- 相談先選定AI (`selectCandidates`) を使って、リクエストに最も合致する相談相手候補をスコア順にカード形式で表示すること
- 各候補のカードに、「推薦理由（AIが判定した合致したトピックなど）」を表示すること
- 相談先候補カードをクリックすると、「選択中」状態になり、日程調整プロセスが活性化されること

## 🛠️ 作業手順
1. `src/app/match/page.tsx` と `src/app/match/match.module.css` を作成する
2. `consult_request` からデータを読み込み、`selectCandidates` で候補リストを取得・表示する処理を実装
3. 美しいホバーアニメーションを伴う相談先カードコンポーネントの構築
4. 選択状態の管理を実装し、次の日程調整セクションに引き継ぐ
