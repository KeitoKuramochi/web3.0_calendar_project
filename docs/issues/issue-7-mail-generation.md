# Issue #7: メール生成・チェック画面の実装 (feature/mail-generation)

## 📌 概要
決定した面談日程・相談相手・件名・形式から、丁寧なメール本文を自動生成し、さらにそのメールの「不足情報（プレースホルダーの残りなど）」や「失礼な表現（フランクすぎる単語など）」を自動チェックする画面を実装します。

## 🎯 ゴール
- `src/app/mail/page.tsx` および `src/app/mail/mail.module.css` の作成
- `consult_match` および `consult_request` からデータを読み込み、`generateEmail` AIを使ってメール下書きを自動生成・表示すること
- ユーザーが生成されたメール本文をテキストエリア内で直接編集できること
- ユーザーの編集に応じて、`checkEmail` AIを自動（またはボタンクリック）で実行し、エラー/警告メッセージと修正案をサイドパネルに表示すること
- 「クリップボードにコピー」ボタンを押したときにコピーされ、コピー完了の視覚効果（トーストなど）が出ること
- 面談調整が完了したことを祝うアニメーション・完了画面を表示できること

## 🛠️ 作業手順
1. `src/app/mail/page.tsx` と `src/app/mail/mail.module.css` を作成する
2. メール生成AI (`generateEmail`) およびメールチェックAI (`checkEmail`) を統合する
3. 本文編集時のリアルタイムバリデーションと、警告パネルのUI設計
4. クリップボードへのコピー機能（`navigator.clipboard.writeText`）の実装
5. 完了モーダルまたは完了表示の実装
