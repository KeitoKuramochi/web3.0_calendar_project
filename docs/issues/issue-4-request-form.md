# Issue #4: 相談リクエスト入力画面の実装 (feature/request-form)

## 📌 概要
「誰かに相談したい」「予定を調整したい」と思ったときに使用するリクエスト入力画面を実装します。ポチポチ入力欄（希望時期、所要時間、対面/オンライン、自分の空き時間）と、自由文入力欄を設けます。

## 🎯 ゴール
- `src/app/request/page.tsx` および関連する CSS Module を作成し、直感的に操作できるリクエストフォームを構築すること
- 以下の入力に対応すること：
  - 相談内容（ポチポチ：トピック選択 ＆ 自由文補足）
  - 所要時間（15分, 30分, 60分, 90分）
  - 形式（対面, オンライン, 指定なし）
  - 自分の空き時間（カレンダーやリスト等での日時選択）
  - 自由文による予定ニュアンス、急ぎ度
- 自由文入力時に、入力汲み取りAI (`parseFreeText`) をリアルタイム（または入力フォーカスアウト時）に呼び出し、ポチポチ入力欄（タイトル、時間、形式など）を自動で推測・補完するシミュレーションが体験できること
- 「相談先を探す」ボタンを押すと、入力内容をローカルの状態に保存して、次のマッチング画面 (`/match`) に遷移すること

## 🛠️ 作業手順
1. `src/app/request/page.tsx` および `src/app/request/request.module.css` を作成する
2. 入力汲み取りAI (`parseFreeText`) との統合ロジックの実装
3. 複数日時を追加できる「自分の空き時間」入力UIの作成
4. 遷移時にセッション情報（ローカルストレージやContext）に入力内容を保存する処理の実装
5. 動作確認を行い、コミット・マージする
